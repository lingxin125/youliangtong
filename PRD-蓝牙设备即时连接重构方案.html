蓝牙电子秤读秤与蓝牙打印小票即时连接方案

本方案将蓝牙电子秤和蓝牙打印机从“重设备管理”调整为“业务场景内即时连接”。用户在开单、暂存、订单详情等场景中直接读秤或打印；我的模块仅保留普通列表轻入口，用于主动连接、断开和切换设备。小票打印覆盖挂单、待开价、待付款和付款成功等节点，付款中 / 处理中 / 已取消不打印小票；小票模板由 PC 后台统一配置并下发到移动端使用。

以下内容用于修正旧方案里的历史口径。评审时以“当前原型口径”为准，旧口径不再作为后续实现依据。

事项旧方案口径当前原型口径评审说明
我的模块入口头像区域曾考虑放独立设备卡片。下方功能列表普通入口，位于“开单配置”下方，顺序为蓝牙电子秤、蓝牙打印机。降低头像区域复杂度，与其他配置类入口保持一致。

3. 背景与目标

业务背景

  • 收购现场需要快速获取毛重、皮重、净重，手工录入容易慢、错、漏。
  • 田间地头收购时，订单可能先暂存为挂单、待开价或待付款，农户需要一张纸质凭证。
  • 旧版“设备管理”思路偏后台维护，现场用户更自然的动作是：点读秤就连接，点打印就连接。
  • 同一收购商可能有多台秤、多台打印机；不同业务员使用的设备可能不同，因此连接状态应以当前手机/当前账号为主。

产品目标

  • 减少现场收粮操作路径，避免用户先去配置设备再回业务页。
  • 开单页只做开单动作，不增加顶部纯状态条和重设备管理入口。
  • 读秤、打印、断开、切换都能在同一个底部抽屉内完成。
  • 蓝牙不可用时不阻断业务，允许手动录入重量或稍后打印。
  • 小票模板由 PC 后台统一控制，移动端只负责按订单状态取模板并发起打印。
  • 形成后续评估可用的标准 PRD，方便技术评估微信小程序 / H5 JSBridge 实现边界。

成功标准

指标标准
操作路径从开单页点击“读秤”到进入读秤确认，不超过一次设备选择。
状态可理解用户能明确知道当前设备是否已连接，并能主动断开。
业务连续性连接失败、断开连接、设备不可用时，不影响手动开单和订单查看。
凭证效率开单暂存成功后可以直接打印对应小票，不必回订单列表查找。

4. 范围与边界

本期纳入

  • 我的模块：下方功能列表中展示蓝牙电子秤、蓝牙打印机普通入口，入口右侧展示连接状态。
  • 连接抽屉：搜索附近设备、连接、主动断开、切换连接、重新搜索。
  • 开单页：读秤按钮触发连接和读数确认，确认后回填重量字段。
  • 多次称重：读秤弹窗保留称重明细，可继续读秤、重新读取、确认回填。
  • 打印小票：暂存后直接打印；订单详情页可打印挂单、待开价、待付款、已完成小票。
  • PC 后台：维护小票模板、字段展示、纸张规格、打印份数和业务状态匹配规则。
  • 异常降级：蓝牙不可用、搜索不到、连接失败、读取失败、打印失败。

本期不纳入

  • 不做重型设备档案管理，不做新增、编辑、启用、停用、删除设备档案作为主流程。
  • 不做商户维度统一分配设备给业务员的复杂权限配置。
  • 不做远程置零、远程去皮、设备参数配置等硬件高级控制。
  • 移动端不提供小票模板编辑能力，模板配置统一放在 PC 后台。
  • 不保证普通浏览器 H5 直接接蓝牙，H5 端需依赖宿主 App JSBridge。

5. 角色与典型场景

角色场景诉求设备动作
收购商超管门店或站点收粮希望设备快速可用，减少现场培训成本。从我的模块功能列表进入蓝牙电子秤或蓝牙打印机，主动连接、断开、切换设备。
收购业务员田间地头收购边称重边开单，暂存后给农户凭证。开单读秤即时连接；暂存后直接打印小票。
农户交粮后等待结算需要知道重量、品类、金额或待确认信息。接收业务员打印的过磅单或订单小票。
技术/实施人员设备适配和现场排障需要明确小程序、H5、硬件协议的边界。按本 PRD 的架构边界评估设备接入方式。

5.2 电子秤读秤流程

用户在开单页点击毛重、皮重或极简快收总重量字段标题右侧的“读秤”。
系统检查当前手机是否有已连接电子秤。
未连接时打开“连接蓝牙电子秤”底部抽屉，展示附近设备。
用户点击连接；已连接设备行展示“已连接 + 断开”。
连接成功后进入读秤确认抽屉，读取稳定重量。
用户可继续读秤、重新读取、删除明细，最后确认回填到目标字段。

5.3 打印小票流程

用户在暂存结果、订单详情或支付成功场景点击“打印小票”。
系统检查当前手机是否有已连接打印机。
未连接时打开“连接蓝牙打印机”底部抽屉，展示附近设备。
用户点击连接；已连接打印机可主动断开或切换。
连接成功后继续打印当前小票，不要求用户重新点击打印。
打印完成后展示轻提示；失败时允许重新打印或重新连接。

7. 小票打印与模板配置方案

打印覆盖节点

订单节点是否打印默认小票类型入口建议说明
挂单支持过磅单 / 收粮暂存单暂存成功页、订单详情、列表更多适用于先拉货后补价或后续结算，给农户留存重量凭证。
待开价支持待开价小票订单详情、列表更多明确重量已确认但价格未定，避免被误解为最终结算单。
待付款支持待付款确认单订单详情、列表更多价格、金额已确认但未完成付款，可作为双方确认凭证。
付款成功支持付款成功小票 / 收购结算小票付款成功页、已结束详情展示最终金额和支付方式，支持后续补打。
付款中 / 处理中 / 已取消不支持不生成小票不展示打印入口付款未形成最终凭证或订单已取消，避免被误解为有效收购凭证；用户可继续完成付款后再打印最终小票。

移动端打印交互

点击打印暂存结果 / 详情 / 付款成功
匹配模板按订单状态取 PC 配置
判断打印机已连接直接打印
未连接弹出蓝牙打印机抽屉
打印结果成功 / 失败 / 重新打印

PC后台模板配置

配置项说明首期建议
模板类型过磅单、待开价小票、待付款确认单、付款成功小票。预置 4 个模板，不建议首期开放无限新增。
字段配置控制小票展示哪些订单字段，例如收购商、订单号、农户、品类、重量、单价、金额、支付方式、操作员、打印时间。字段可勾选展示，关键字段可设为必选不可取消。
打印规则按业务节点配置默认模板、是否自动打印、默认打印份数、是否允许补打。首期以手动打印为主，付款成功可评估自动打印开关。
纸张规格支持 58mm / 80mm 热敏纸模板预览。先以 58mm 为默认，80mm 作为后续兼容项。
模板发布PC保存后形成生效配置，移动端打印时读取当前生效模板。保存即生效；后续可补版本号和发布记录。

原型入口：PC 后台小票模板配置。该页面只表达后台配置模型，不改变移动端现场打印的轻量交互。

10. 业务规则

设备连接规则

  • 同一类型设备同一时间只保留一个“当前已连接”。
  • 连接另一台同类型设备时，原设备自动变为未连接。
  • 已连接设备必须支持主动断开；断开后不强制跳转页面。
  • 连接状态以当前手机终端为准，不代表整个收购商主体都已连接。
  • 历史设备用于快速识别和重连，但仍需设备实际在线。

读秤与打印规则

  • 电子秤只提供当前重量，订单归属、累加、回填由丰收云页面完成。
  • 多次称重必须保留明细，不能只把多次读数直接累加后丢失记录。
  • 读秤回填后触发现有毛重、皮重、净重、金额计算逻辑。
  • 过磅单适用于挂单 / 待开价等未付款场景；订单小票适用于待付款 / 已完成等场景。
  • 打印失败不改变订单状态，只提示用户重试或重新连接打印机。
  • 移动端不编辑小票字段，字段、纸张和打印份数以 PC 后台生效配置为准。

小票模板规则

规则说明
模板匹配移动端按订单状态匹配模板：挂单=过磅单，待开价=待开价小票，待付款=待付款确认单，付款成功=付款成功小票。
关键字段订单号、收购商、农户、农产品、净重等建议作为必备字段；金额和支付方式仅在适用模板中展示。
补打口径订单详情支持补打当前订单对应的小票；付款中、处理中、已取消不展示打印入口。
自动打印首期以手动打印为主，自动打印作为 PC 后台开关配置项，避免现场重复出纸。

订单状态与小票类型规则

订单状态是否可打印小票类型入口关键限制
挂单可打印过磅单暂存结果、列表、详情 / 编辑页不展示最终付款信息。
待开价可打印待开价小票列表、详情 / 编辑页强调价格未定，不作为最终结算凭证。
待付款可打印待付款确认单列表、详情 / 编辑页可展示金额，但不展示付款成功信息。
付款成功可打印付款成功小票付款成功页、已结束详情展示最终金额、支付方式和打印时间。
付款中 / 处理中不可打印无不展示入口支付结果未确认，避免形成误导凭证。
已取消不可打印无不展示入口订单已失效，不再出具小票。

设备记录归属

维度建议口径理由
设备可见性收购商主体下可识别同类设备，但首期不做完整设备台账。方便后续扩展多业务员、多设备，但当前不增加管理复杂度。
连接使用当前登录账号 + 当前手机终端。蓝牙连接本身发生在本机，业务员 A 和 B 可分别连接自己的设备。
历史记录本地缓存最近连接设备，可用于下次快速连接。减少重复搜索和选择成本。

11. 异常与降级

异常场景页面表现降级路径
蓝牙未开启连接抽屉提示开启蓝牙，保留重新搜索。用户开启蓝牙后重搜；开单可手动录入。
未授权附近设备 / 定位提示授权用于扫描附近蓝牙设备。授权后重试；不授权时手动录入或稍后打印。
搜索不到设备列表为空，提示确认设备开机、靠近手机、未被占用。重新搜索；手动录入重量；订单后续再打印。
连接失败设备行保持未连接，Toast 提示失败原因。重新连接或选择其他设备。
连接中断当前设备状态刷新为未连接；读秤或打印动作失败。重新连接后继续读秤 / 打印；已填写数据不清空。
读数不稳定读秤确认抽屉展示不稳定状态，不自动回填。等待稳定、重新读取或手动录入。
读数为 0 或为空确认回填前提示请先读取有效重量。重新读取；仍异常时手动录入。
打印失败提示打印失败，保留当前订单和小票类型。重新打印、断开重连、稍后到订单详情打印。
打印机缺纸 / 卡纸若硬件返回错误码，提示检查纸张或打印机状态。处理硬件后重新打印；订单状态不变化。
模板配置缺失使用系统默认模板并提示后台配置待完善。不阻断打印；后台后续补齐模板。
H5 无 JSBridge提示当前环境暂不支持蓝牙设备。切到小程序 / 宿主 App；业务继续走手动操作。

12. 验收标准与测试用例

12.1 验收标准

  • 我的模块设备入口展示在下方功能列表内，位于开单配置下方，入口右侧状态可同步刷新。
  • 我的模块、开单页、订单详情页的设备连接抽屉交互一致。
  • 已连接设备行必须能主动断开，断开后入口状态和列表状态同步刷新。
  • 点击读秤时，如果未连接电子秤，必须先进入连接抽屉；连接成功后继续读秤。
  • 暂存成功后可直接打印小票；订单详情页也可再次打印。
  • PC 后台可配置小票模板字段、状态规则和预览，移动端按订单状态匹配对应模板。
  • 蓝牙异常不阻断手动开单、订单查看、继续付款等主业务流程。
  • 付款中 / 处理中 / 已取消不展示打印小票入口。
